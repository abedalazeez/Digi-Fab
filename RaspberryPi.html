LDR using photoresistor and Capacitor

As we know the LDR sensor is an analog sensor so we cant use directly with raspberry pi one easy solution is a use capacitor so we can calculate the charging time for our capacitor and that will give us a logical view of the light-dependent resistor so by using resistor LDR capacitor circuit we make a converter from analog to digital and we have an overview of the light or brightness
